# example_am

## Background

A repository to show some data validation, training and testing.

The dataset that has been used is available [here][dataset]

## Usage
This repo uses uv package and project manager, installation instructions can be found [here][uv]

Once uv is installed, using the command line, navigate to this repo. The data validation code can be ran using ```uv run python data_validation.py```. The training code can be ran using ```uv run python train.py``` and the testing code can be ran using ```uv run python test.py```.

[dataset]: https://osf.io/myfu8/metadata/osf
[uv]: https://docs.astral.sh/uv/
